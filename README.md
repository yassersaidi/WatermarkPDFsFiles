# WatermarkPDFsFiles
Add watermark to a pdf file with Python
## PyPDF2
install the PyPDF2 using PIP:
```bash
pip install PyPDF2
```
## Run
run the file using terminal
```bash
python waternark_pdf.py super_file watermark_file output_file
```
**Example**:
```bash
python waternark_pdf.py file1.pdf file2.pdf output.pdf
```
